Introduction

Developed by CyberArk Labs, ByteCodeLLM is a state-of-the-art decompilation framework for converting Python bytecode, up to version 3.13, back to source code.
Utilizing local large language models (LLMs), it offers unparalleled accuracy and operates entirely on-premise, enhancing both privacy and efficiency.
This makes ByteCodeLLM ideal for developers, reverse engineers, and security professionals seeking a reliable tool for reverse engineering and source code recovery without the need for external services.

How It Works?

Note: Decompilation completeness can vary. Follow these guidelines to enhance success rates:

  • ByteCodeLLM is optimized for Python 3 bytecode but plans to support other languages in the future.
  • Supported file types include: PYC, EXE (planned), Folder of PYC files (planned).
  • The tool uses pycdc and pycdas for processing, ensure these are installed and paths are configured as per the installation guide.
  • For enhanced decompilation, the LLM can be configured to run locally or remotely, adjusting to your privacy and performance needs.
  • Use the "Separated Files" checkbox if working with multi-file directories to handle each file independently.

Technical Information

ByteCodeLLM specializes in decompiling Python bytecode files back to their original source code using advanced local large language models (LLMs). The tool operates by first utilizing pycdc and pycdas to extract the bytecode and generate an initial, often incomplete, source code reconstruction.
For sections of code that pycdc cannot fully decompile, ByteCodeLLM employs finetuned LLMs to interpret the bytecode and fill in the gaps, improving the accuracy and completeness of the decompiled source. This process ensures that even complex or obfuscated Python code can be accurately reconstructed, preserving the original logic and structure as much as possible.
